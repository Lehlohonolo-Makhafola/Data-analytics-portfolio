# Healthcare Patient Analysis Dashboard

## Project Overview
Analysis of 54,966 patient records from a 
healthcare dataset covering medical conditions, 
billing amounts, insurance providers, admission 
types and treatment outcomes.

## Tools Used
Microsoft Excel — Data Cleaning, Pivot Tables,
Dashboard Design, DAYS Formula, PROPER Formula

## Dataset Source
Kaggle — Synthetic Healthcare Dataset

## Data Quality Issues Found and Fixed
- Corrupt hospital names replaced via imputation 
  with 10 standardized SA hospital names
- Negative billing amounts converted to positive
- Patient name titles removed using Find and Replace
- Inconsistent name casing standardized using 
  PROPER formula
- Billing outliers under R100 flagged using 
  helper column
- Days Admitted column created using DAYS formula

## Key Insights
- Cigna and Medicare carry highest patient volume
- Arthritis and Diabetes are most prevalent 
  conditions across all age groups
- Emergency admissions generate highest 
  average billing amounts

## Project Structure
- Raw Data — original unmodified Kaggle dataset
- Cleaned Data — transformed dataset with 
  full Change Log
- Pivot Tables — analysis across all key metrics
- Dashboard — final visualization and insights
